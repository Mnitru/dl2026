\section{Introduction}
This report presents an implementation of gradient descent from scratch to minimize the function:
\[
f(x) = x^2
\]

The derivative is:
\[
f'(x) = 2x
\]

\section{Algorithm}
The update rule used is:
\[
x = x_A - r \cdot f'(x_A) = x - 2rx_A
\]

where $r$ is the learning rate.

\section{Implementation}
The algorithm is implemented in Python without using any libraries. The process iteratively updates $x$ until convergence or until a maximum number of iterations are reached.
